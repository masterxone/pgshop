Fitur & Keunggulan
Intelligent Caching
Cache in-memory 15 detik dengan single-flight lock. Mencegah request berulang ke server eksternal saat trafik tinggi.
IP Rate Limiting
Limit 150 request per 15 detik per IP. Trafik mencurigakan diblokir otomatis selama 5 menit.
Auto Fallback
Otomatis menyajikan data cache terakhir jika API eksternal mengalami gangguan.
Zero Persistence
Kredensial hanya diproses di RAM. Tidak ada token atau username yang ditulis ke disk.
Source Tracking
Metadata asal data (Fresh API, Valid Cache, atau Fallback) disertakan untuk pemantauan real-time.
Domain Agnostic
Endpoint terdeteksi otomatis sesuai host aktif. Tidak perlu ubah kode saat ganti domain.
Autentikasi Keamanan
Wajib menyertakan parameter username dan token (userId:secret) dalam body request metode POST. Jangan bagikan token Anda kepada siapapun.
Dapatkan Token OrderKuota
Login dengan username dan password akun OrderKuota. OTP dikirim ke email terdaftar, lalu masukkan OTP untuk mendapatkan token.
